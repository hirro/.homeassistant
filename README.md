  * [Telldus z-wave plus](http://telldus.se/wp-content/uploads/311912_Quick_Guide_160804_Web.pdf)

  * [NEXA LMDT-810 (433MHz)](https://www.nexa.se/vara-produkter/system-nexa/sensorer/lmdt-810-rorelsevakt-utomhusbruk)
